About Base64 Encoding

What is Base64?
Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format.
Common Uses
Used for encoding data in emails, storing complex data in XML or JSON, and embedding images in HTML/CSS.
Encoding Process
Converts every 3 bytes of binary data into 4 ASCII characters from a set of 64 different characters.
